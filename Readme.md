# 🍽️ Restaurant Order System (CLI)

Sistema de gerenciamento de pedidos de restaurante desenvolvido em Python, utilizando Programação Orientada a Objetos (POO).

## 📌 Funcionalidades

* 📋 Exibição do cardápio
* 🧾 Criação de pedidos por cliente
* ➕ Adição de pratos ao pedido
* ➖ Remoção de pratos do pedido
* 📦 Controle de estoque dos pratos
* 💰 Fechamento de pedido com cálculo do total

## 🛠️ Tecnologias utilizadas

* Python
* Programação Orientada a Objetos (POO)

## 🧠 Conceitos aplicados

* Classes e objetos
* Encapsulamento
* Listas e manipulação de dados
* Tratamento de erros (`try/except`)
* Organização modular de código

## 📂 Estrutura do projeto

```
📁 modelos/
 ├── prato.py
 ├── cliente.py
 └── pedido.py

main.py
```

## ▶️ Como executar

1. Clone o repositório:

```
git clone https://github.com/LePignatari/sistema-pedidos-restaurante.git
```

2. Acesse a pasta:

```
cd sistema-pedidos-restaurante
```

3. Execute o projeto:

```
python main.py
```

## 💡 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar lógica de programação e conceitos de POO, simulando um sistema real de pedidos em restaurante via terminal.
